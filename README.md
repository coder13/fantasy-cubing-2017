# Fantasy Cubing

http://fantasy.cubing.net

Tech:

- Hapi 
- MySQL
- Redis (cache)
- Webpack
- React
- AmpersandJs
- Semantic-UI


## Setup:
- Install nodejs
- `git clone https://github.com/coder13/fantasycubing`

## Development:
Run both the dev server and webpack:

- `npm run dev-server`
- `npm run webpack`
